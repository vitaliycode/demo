О SmolVLM Demo

Возможности

  • Visual Question Answering (VQA): Задавайте вопросы об изображениях и получайте интеллектуальные, контекстно-зависимые ответы
  • OCR (Распознавание текста): Извлекайте текст из изображений с высокой точностью
  • Описание изображений: Автоматически генерируйте описания для ваших изображений
  • Мультивопросные диалоги: Продолжайте задавать вопросы об одном и том же изображении
  • Экспорт результатов: Скачивайте результаты в виде текстовых файлов

Как использовать

  1. Выберите функцию на вкладках выше
  2. Загрузите изображение, нажав или перетащив его в область загрузки
  3. Для VQA: Введите ваш вопрос и нажмите "Задать вопрос"
  4. Для OCR: Нажмите "Извлечь текст" для распознавания текста на изображении
  5. Для описания: Нажмите "Создать описание" для генерации описания
  6. Скачайте или скопируйте результаты по необходимости
