# SmolVLM Demo

**Мультимодальная система искусственного интеллекта для работы с изображениями**

## Быстрый старт

```bash
# CPU режим
./start.sh cpu

# GPU режим  
./start.sh gpu
```

Затем откройте:
- **Веб-интерфейс**: http://localhost:8000
- **Gradio UI**: http://localhost:8000/test

## Возможности

- **Visual Question Answering** - Задавайте вопросы об изображениях
- **OCR** - Распознавание текста
- **Image Captioning** - Автоматическое описание
- **Мультитурновый диалог** - Несколько вопросов об одном изображении

## Требования

- Docker и Docker Compose
- 8GB RAM (CPU) или 6GB VRAM (GPU)
- 10GB свободного места
